# Contributing

This repository is currently in read-first mode.

- Bug reports are not accepted yet.
- Feature requests are not accepted yet.
- Pull requests may be closed without review.
- Public design discussion is planned for `between-docs` after the documentation path is ready.

For security matters, please do not use public channels.
Please follow `SECURITY.md`.

Please use this repository primarily as a reference source.

Thank you for your interest.
